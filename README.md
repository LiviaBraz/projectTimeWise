<h4> Descrição: </h4>
O projeto TimeWise tem como objetivo desenvolver uma plataforma de gerenciamento de tempo e tarefas que ajuda os usuários a otimizar sua produtividade pessoal e profissional. A plataforma fornecerá recursos avançados de acompanhamento do tempo, organização de tarefas e insights valiosos para melhorar a gestão do tempo.

<h4> Benefícios e Valor: </h4>
O TimeWise oferecerá aos usuários a capacidade de:

- Acompanhar e registrar o tempo gasto em várias atividades.
- Definir metas de produtividade.
- Criar e gerenciar listas de tarefas de forma eficaz.
- Visualizar relatórios e análises para identificar oportunidades de melhoria na gestão do tempo.
- Aumentar a eficiência e o equilíbrio entre trabalho e vida pessoal.

<h4> Público-Alvo: </h4>
O público-alvo do TimeWise inclui:

- Profissionais autônomos que desejam rastrear o tempo gasto em projetos e tarefas.
- Equipes de trabalho que buscam uma solução de gerenciamento de tarefas colaborativa.
- Estudantes que desejam melhorar sua produtividade acadêmica.
- Qualquer pessoa interessada em melhorar sua gestão do tempo e alcançar seus objetivos de forma mais eficiente.

<h4> Escopo do Projeto: </h4>
O escopo do projeto inclui o desenvolvimento de:

- Sistema de Login 
- Pagina Inicial com: 
    > Agenda
    > Habit Tracker 
- Controle de ponto de atividades por categoria
- Página para visualização com dashboards 
- Página para controle financeiro 

<h4> Funcionalidades extras: </h4>

- Configurações para o usuário (modo noturno, preferencias de notificação, fuso horario) 
- Notificações e lembretes 
- Gerar relatórios sobre o uso do tempo e desempenho das tarefas 
- Documentação para ajuda ao usuário (sessão help)

<h4> Tecnologias Utilizadas: </h4>
O projeto TimeWise utilizará as seguintes tecnologias:

- Frontend: Angular
- Backend: Spring Boot Java
- Banco de Dados: MySQL
- Integração: APIs - Calendários, Tarefas

<h4> Riscos e Desafios: </h4>
Alguns desafios potenciais incluem:

- Garantir a segurança dos dados do usuário.
- Manter uma interface simples e intuitiva.
- Garantir a escalabilidade da plataforma conforme o número de usuários aumenta.
